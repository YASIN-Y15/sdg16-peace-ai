## UN Sustainable Development Goal 16
**Promote peaceful and inclusive societies for sustainable development**

## Problem Statement
Political instability and social conflicts threaten global peace. This AI system predicts regions at high risk of conflict using machine learning, enabling early intervention and peace-building initiatives.

## Machine Learning Approach
- **Algorithm**: Random Forest Classifier
- **Type**: Supervised Learning - Binary Classification
- **Features**: Youth unemployment, Social inequality, Education access, Political polarization, Hate speech levels, Previous conflicts, Economic growth, Media freedom
- **Target**: Conflict risk (High/Low)
